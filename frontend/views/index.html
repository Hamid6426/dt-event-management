API Documentation

Event API Endpoints

  • GET /api/v3/app/events - Get all events or query by ID
  • GET /api/v3/app/events/:id - Get a single event by ID
  • GET /api/v3/app/events?type=latest&limit=5&page=1 - Get latest events with pagination
  • POST /api/v3/app/events - Create a new event
  • PUT /api/v3/app/events/:id - Update an existing event
  • DELETE /api/v3/app/events/:id - Delete an event by ID
